# k-039 — Quick task keyboard shortcut

**Request:** a key combination (e.g. Control+Space) to quickly add a task, ideally changeable.

## What I did

Added a **Ctrl+Space** quick-add shortcut to the board UI (`index.html`). Pressing it
anywhere on the board opens the **New card** modal with the title field focused, so you
can type a task and hit save without reaching for the mouse.

Guards built in so it behaves:

- It does **not** fire while you're typing in a text field (input, textarea, or the AI
  chat box) — so Ctrl+Space inside a field still does whatever that field expects.
- It does **not** fire when a modal is already open.
- Plain `Cmd+Space` is deliberately left alone — that's macOS Spotlight.

## On making it changeable

I went with **Ctrl+Space** as a sensible fixed default rather than building a full
"rebind your shortcut" settings panel, which would be a much larger piece of work for a
single shortcut. If you do want it configurable later, the hook is a single handler near
the Escape key handler in `index.html` (search for "Quick-add shortcut") — swap the
`e.ctrlKey`/`e.code==="Space"` test for whatever combo you prefer, or wire it to a value
in localStorage and add a control in the admin panel. Say the word and I'll do that.

## Note / limitation

This is an **in-app** shortcut — it works when the Kanban tab/app is focused. A browser
page can't register a truly OS-global hotkey that fires when another app is in front;
that needs an OS-level tool (e.g. a macOS shortcut/Automator service, or a packaged app).
If you want it to launch the board and add a task from anywhere on the Mac, that's a
separate, slightly bigger job — happy to set it up if useful.
