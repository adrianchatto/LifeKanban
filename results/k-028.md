# k-028 — Spin-wheel Pomodoro (set any duration)

**Done.** The Pomodoro timer is no longer locked to 25 / 5 / 15. The time is now a spin wheel you can dial to any number of minutes, one minute per step.

## How to set the time

Hover the big time display and dial it in whichever way feels natural — they all move in 1-minute steps:

- **Scroll** up/down over the timer.
- **Drag** up/down on it.
- Click the **▲ / ▼** chevrons above and below the number.
- Focus it and use the **arrow keys**.

A small hint under the timer shows the current setting (e.g. "40 min · spin to change"). The range is 1 to 180 minutes.

## Details worth knowing

- **Each phase keeps its own length.** Focus, Short and Long still exist as tabs, but you can now dial each one to whatever you like — switch to a tab, spin it, and that length sticks. So you could run, say, 50-minute focus blocks with 10-minute breaks, or anything else.
- **Your settings are remembered** between sessions, like the rest of the timer state.
- **Dialing is disabled while the timer is running**, on purpose — so an accidental scroll over the widget can't disrupt a running countdown. Pause (or let the phase finish) to change the length, then start again. The chevrons hide while running to make this clear.
- The gear settings panel is still there if you'd rather type exact numbers; its limits were raised to match the wheel (up to 180 minutes).

Everything else about the timer — start/pause, reset, skip, the 🍅 count, break auto-advance, the chime and notifications — is unchanged.
